This is a sample food blog post. When you want to add a new post, simply create a new markdown file in this folder with the same frontmatter format.

## How to Add New Posts

1. Create a new `.md` file in `src/content/blog/food/`
2. Add the frontmatter at the top (title, description, date, image, tags)
3. Write your content in markdown below the frontmatter
4. The post will automatically appear on the Food Blog page

## Markdown Features

You can use all standard markdown features:

- **Bold text** and *italic text*
- Lists like this one
- [Links to other sites](https://example.com)
- Images and more
